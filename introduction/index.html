Repository¶

Project Repository¶

Github URL: https://github.com/stabilaprotocol.

  • stabilaprotocol/java-stabila is the source code of the MainNet.
  • stabilaprotocol/protocol is the defination of the api and data structure.
  • stabilaprotocol/wallet-cli is the official command line wallet.
  • stabilaprotocol/tips is the STABILA Improvement Proposals.
  • stabilaprotocol/unit-network is a project dedicated to building a trustful decentralized sidechain of STABILA Main Chain.

Configuration File for the Net¶

MainNet Configuration: GH stabilaprotocol/StabilaDeployment/main_net_config.conf

TestNet Configuration: GH stabilaprotocol/StabilaDeployment/test_net_config.conf
